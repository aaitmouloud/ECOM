Description

VVVVVV est un jeu de plateforme en 2D de Terry Cavanagh, créateur de douzaines de jeux gratuits. Vous êtes le leader d'une équipe de scientifiques qui explorent des dimensions et qui se retrouvent, après un crash, séparés. Vous allez devoir explorer ce monde étrange et retrouver vos amis. VVVVVV ne vous permet pas de faire des sauts. Vous pourrez cependant inverser la gravité en appuyant sur une touche. Le jeu est d'une grande liberté d'évolution : il n'y a pas d'endroits fermés, de pouvoirs ou de leviers.
